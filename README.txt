Delivered requirements:
1. User can plan for the recipe of a week.
2. User can save each recipe.
3. User can know the statistical data of what he eats this week.
4. User can login the app.
5. User can know the raw materials that he needs for a week.
6. User can send picture for recipe.
8. The app can be networked.
